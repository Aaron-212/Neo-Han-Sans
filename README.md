## Roadmap 规划

- 预估在5月底～6月初完成GB2312的6763个汉字；
- 尝试将UPM从1000提到2048；
- 尝试让Light母版更细；
- 待所有GB2312汉字改完后，会添加中文标点符号，并将尝试支持直排标点；

注意：
- 对于GB2312外的汉字，暂时没有打算添加。可能会根据中国大陆汉字通用规范一级至三级逐步添加；
- 只会有符合中国大陆标准的字形；
- 暂时不会添加拉丁字母、西里尔字母、希腊字母、日本假名、朝鲜文等其他非汉字字母。因为西文部分可能会直接用Inter。
